# OpenCode Walkthrough

A Visual Studio Code extension that provides an interactive walkthrough to help you get started with [OpenCode](https://opencode.ai) — a CLI tool that assists with software engineering tasks directly from your terminal.

## Features

- **4-step guided walkthrough** inside VS Code's native Getting Started experience
- Covers installation, inline prompts, interactive sessions, and best practices
- Launch anytime via the Command Palette

## Requirements

- [VS Code](https://code.visualstudio.com/) 1.74+
- [OpenCode](https://opencode.ai) (CLI) — walkthrough covers installation

## Usage

1. Open the **Command Palette** (`Cmd+Shift+P` / `Ctrl+Shift+P`)
2. Run **"OpenCode: Show Walkthrough"**
3. Follow the step-by-step guide

The walkthrough is also available in VS Code's **Get Started** page under the walkthroughs section.

This extension contributes no settings.
