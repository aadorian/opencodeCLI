## Add an MCP Server

MCP (Model Context Protocol) servers extend OpenCode with external tools —
browsers, databases, search engines, ticketing systems, and more.

Run the interactive setup to register your first server:

[Add MCP Server](command:opencode-walkthrough.addMcp)

The CLI will ask whether your server is **local** (a subprocess) or
**remote** (an HTTPS endpoint). The server is saved to `opencode.json`
automatically once configured.

To see all connected servers:

[List MCP Servers](command:opencode-walkthrough.listMcp)

Learn more: [opencode.ai/docs/mcp-servers/](https://opencode.ai/docs/mcp-servers/)
