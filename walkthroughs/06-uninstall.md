## Remove OpenCode

If you no longer need OpenCode, you can uninstall it from your system using the command below.

You will be asked whether to keep your configuration and session data, or remove everything.

### From the terminal

```bash
opencode uninstall
```

To keep your config and data while removing the binary:

```bash
opencode uninstall --keep-config --keep-data
```

### If installed via npm

```bash
npm uninstall -g opencode-ai
```

### If installed via Homebrew

```bash
brew uninstall opencode
```
