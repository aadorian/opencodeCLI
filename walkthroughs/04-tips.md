### Use the terminal in VS Code

OpenCode runs in your terminal. Press `` Ctrl+` `` (`` Cmd+` `` on macOS) to open the integrated terminal.

### Be specific

Include the language, framework, and expected behavior in your prompts. The more context you provide, the better the results.

### Iterate

Refine results with follow-up prompts:

```bash
opencode "now add error handling to that function"
```

### Check the docs

Full documentation is available at [opencode.ai](https://opencode.ai).
