OpenCode Walkthrough

A Visual Studio Code extension for OpenCode — walkthrough onboarding, sidebar trees, CLI quick actions, and an in-editor agent harness.

Install from Marketplace View on GitHub OpenCode CLI docs

Features

Getting Started walkthrough

Step-by-step onboarding for install, inline prompts, interactive mode, and custom agents.

Sidebar trees

Browse agents, models, MCP servers, and sessions from the OpenCode activity bar.

Command palette & shortcuts

Run OpenCode CLI commands from the status bar, quick pick, and keyboard shortcuts.

Agent harness

VS Code–native agent loop with context assembly, tools, and chat webview (in progress).

Screenshots

Quick install

Install the OpenCode CLI, then add the extension from the Marketplace or clone this repo and press F5 in VS Code.

npm install -g opencode-ai
opencode --version
